# The commands used in the project

```
    go mod init github.com/somashekhar/snippetbox
    go run .

    # Download/Verify pkg in go.mod
    go mod download
    go mod verify

    # Install/uninstall/clean packages
    go get -u github.com/foo/bar
    go get -u github.com/foo/bar@none
    go mod tidy -v 

    go test 
    go build 
```

Stop the server using Ctrl+c
